# Phase 27 — `--help` stars a range default that matches a sentinel

*Realizes design Decision 12 (amended: a `ReasoningRange` default that equals a listed sentinel is starred in place and the trailing `default …` clause is dropped, exactly as an enum value is; a slot-less range default keeps its trailing clause). Depends on Phase 26 (`WriteHelp`, its `Kind`-keyed render routine's `ReasoningRange` branch, and its golden).*

Extend the `ReasoningRange` branch of the render routine inside `repl.WriteHelp` so a range default that coincides with a listed sentinel is marked in place instead of spelled in a trailing clause. When `spec.Default` equals one of the range's sentinel values, that sentinel's rendered token is prefixed with `*` (`v=*meaning`) and **no** trailing `default …` clause is printed: `thinking_budget=<0–24576>  (0=off, -1=*dynamic)` and `thinking_budget=<128–32768>  (-1=*dynamic)`. When the default matches no sentinel — a slot-less `off` (no `0=off` sentinel, e.g. `claude-haiku-4-5`) or a plain in-range budget — the trailing clause is kept unchanged (`(default off)`). This applies to the `ReasoningRange` branch only the same in-place rule the enum/toggle rows already follow; the native default token is still computed by the shared formatter (Decision 3) in both the starred and the trailing-clause case. The `Term`-drop from Phase 26, the enum/toggle branches, the `<Min>–<Max>` value syntax, `termToKey`, the `(no reasoning control)` fallback, the credential-blind contract, and `Default()`-order grouping are all unchanged. The change is observable only in the rendered bytes, so the `WriteHelp` golden under `testdata/` is regenerated: the two Gemini range rows gain the starred sentinel and lose their `; default dynamic` clause, and `claude-haiku-4-5` stays `(default off)`.

**Done when:** the following is covered by clearly-named tests (the regenerated `WriteHelp` golden plus a focused assertion on the range rows) and the suite is green:
- R-APDX-FP3D — a `ReasoningRange` whose `spec.Default` equals a listed sentinel marks that sentinel in place with a `*` and prints no trailing `default …` clause (`thinking_budget=<0–24576>  (0=off, -1=*dynamic)`, `thinking_budget=<128–32768>  (-1=*dynamic)`); a range whose default matches no sentinel — a slot-less `off` or a plain in-range budget — keeps its trailing `(default <d>)` unchanged (`thinking_budget=<1024–max_tokens>  (default off)`).
